Changes from 0.3 to 0.4
-----------------------

Unlimited length scan lists

Extra user flag ABEND shows '1' if user did not log off normally last visit.

Extra user flag NODEGROUP shows nodegroup number of user's current terminal.

Stopped distributing with sample menus from the live devel site as they were
just confusing new sysops!

new menu action named "check_interval" to stop people logging on again too
soon after last time.
